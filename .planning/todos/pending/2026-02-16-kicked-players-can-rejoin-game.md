## Problem

Сейчас если админ кикает игрока:
1. Игрок видит модалку "Вас удалили из игры"
2. При попытке вернуться по ссылке — снова видит эту ошибку
3. Не может заново присоединиться к игре

Это слишком жёсткое поведение. Кик должен выкидывать из текущей сессии, но не блокировать повторное подключение.

## Solution

1. При кике — очищать localStorage сессию для этой игры
2. Редиректить на /game/[id]/join вместо показа ошибки
3. Не хранить "kicked" статус в БД — просто удалять игрока
4. Игрок может заново ввести имя и выбрать аватар
5. Модалка "Вас удалили" показывается один раз, потом редирект на join
